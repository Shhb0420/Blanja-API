## Getting Started
Berikut panduan untuk menjalankan project ini secara lokal
### How to install

* Clone Repository
    ```sh
    https://github.com/Shhb0420/Blanja-API.git
    ```
* ExpressJS
    ```sh
    npm i express
    ```
* MYSQL
    ```sh
    npm i mysql
    ```
* Morgan
    ```sh
    npm i morgan
    ```

## Endpoint
1. Get
    ```sh
        * /products
        * /products/byPrice
        * /products/byDate
        * /product/{id}
        * /search
        * /history
    ```

2. Post
    ```sh
        * /products
        * /transaksi
    ```

3. Update
    ```sh
        * /products-update
    ```

4. Delete
    ```sh
        * /products/delete/{id}
    ```
